企业级客户申请通道
一、延迟退房
1、正常退房时间为12:00，如退房当日无预定，退房时间可延迟到14:00；
2、如退房当日房源被他人预定，且预定客人入住时间在15:00及之前，则无法提供延迟退房服务；
二、行李寄存
1、如需行李寄存请提前联系对应的助理房东（行李寄存时间及寄存服务点请以各城市助理房东提供为准）；
2、仅限退房当天享受免费寄存，第二天则需额外收费；
3、行李寄存为自主服务，客人如有寄存需要，需自行将行李送往行李寄存服务点；
4、贵重物品不寄存，任何易燃、易爆、放射性、腐蚀性物品、剧毒物品、毒品、武器以及其他国家禁止存放的违禁物品不予寄存；
三、入住接待
1、需要提前跟助理房东确定好入住时间（入住接待时间为15:00至19:00），并提出需要入住接待服务；
2、确定好的入住时间如需更改，需要入住前一天更改，当天临时更改，有可能会导致入住接待服务无法正常提供。
